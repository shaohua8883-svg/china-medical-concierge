# China Medical Concierge

A premium medical concierge website for foreigners coming to China for healthcare.

## Features

- **Hospital & Doctor Matching** - Connect with top Chinese hospitals
- **Appointment Booking** - No waitlists, no language barriers
- **Medical Translation** - Certified interpreters at every appointment
- **Travel & Visa Support** - Full door-to-door assistance
- **Post-treatment Follow-up** - Bilingual records and remote consultations
- **Live Chatbot** - AI-powered virtual assistant (Warren)
- **Contact Form** - Free case review submissions
- **Cost Comparison** - Compare treatment costs across countries
- **Responsive Design** - Works on all devices

## Deploy to Render

1. Push this code to GitHub
2. Go to [render.com](https://render.com) and sign up for free
3. Click "New Web Service"
4. Connect your GitHub repository
5. Use these settings:
   - **Build Command:** `npm install` (or leave empty since we have no dependencies)
   - **Start Command:** `node server.js`
   - **Environment:** Node
6. Click "Create Web Service"
7. Your site will be live at `https://your-app-name.onrender.com`

## Deploy to Railway

1. Push this code to GitHub
2. Go to [railway.app](https://railway.app) and sign up for free
3. Click "New Project" → "Deploy from GitHub repo"
4. Select your repository
5. Railway will auto-detect Node.js and deploy

## Local Development

```bash
node server.js
```

Then open `http://localhost:3000` in your browser.
